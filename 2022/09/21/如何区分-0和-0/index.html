如何区分-0和+0

创建时间：2022-09-21

更新时间：1 分钟前

方案一：使用 ES5 的语法 Object.is

var a = -0;
Object.is(a, +0); // false
Object.is(a, -0); // true

方案二： 用非 0 的数字去除以 0、-0 和 + 0，会分别得到 Infinity、-Infinity 和 + Infinity

var b = -0;
1 / b === -Infinity; // true
var c = +0;
1 / c === +Infinity; // true
var d = 0;
1 / d === Infinity; //true
