MacBook M1 安装安卓模拟器

创建时间：2023-02-13

更新时间：1 分钟前

作为一个前端开发，适配安卓机那是必须得考虑的，没有安卓手机，在电脑上装一个安卓模拟器也是个不错的选择。坏消息是，在 M1 芯片笔记本上，夜神、mumu 、genymotion 等等都不好用，辛辛苦苦安装好，结果模拟器一直打不开。好消息是谷歌专门对 M1 机型做了一个安卓模拟器。

  • 下载地址：https://github.com/google/android-emulator-m1-preview/releases/tag/0.3 (opens new window)

  • 备用地址：https://pan.baidu.com/s/1qK8i6QkhSZTXcAM-y5Mtuw?pwd=wb4u (opens new window)

常见问题：

  1. 刚打开模拟器就提示我 ADB 目录没有检测到。

    解决：点击软件右下角的... 图标，进入 Setting 标签页，关闭 Use detected ADB location，点击文件夹图标，定位到 ADB 安装目录。

  2. 把 apk 拖进模拟器，提示没有找到 ADB，无法安装。

    解决：操作同上，不用重启，重新把 apk 拖进模拟器。

  3. 安装完 apk 后，在模拟器桌面看不到。

    解决：鼠标向上滑动模拟器屏幕，就能看到所有已安装的 app。

  4. 模拟器自带的 chrome 浏览器打不开。

    解决：官方也知道这个问题，但一直没修复。放弃模拟器自带的 chrome 浏览器，自己装一个浏览器。
